Transcriptions - a text editor for fast manual transcription on the Mac.

Requirements: MacOS 12 Monterey or later


Used around the world in research, education and journalism since 2008.
MacBreak Weekly pick of the week on March 30, 2010.

Features

* Media with tape behavior (including variable playback speed)
* Customizeable media-control shortcuts (NOW with Fast Forward)
* Timestamps
* Text substitution
* Footpedal-support
* Subtitle Editing

Load media in the player ("File" - "Open media file..."), use the media-controls to adjust speed and volume, set keyboard shortcuts for media control in the "Preferences" pane and type away. Transcriptions also supports text-substitutions by inserting "@" followed by a text-token. Insert timestamps into the text to jump to a particular position in the loaded media file.
Check out the App Overview to get started.
